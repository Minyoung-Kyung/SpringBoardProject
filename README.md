# SpringBoardProject
> Spring CRUD

## 🛠 기능 🛠 

* 게시물 목록, 상세보기
    * 글번호
    * 제목
    * 작성자
    * 작성일 : 오늘이면 'HH:mm:ss', 오늘이 아니면 'yyyy-MM-dd'
* 게시물 작성
    * 제목
    * 이름
    * 비밀번호
    * 내용
* 게시물 수정
    * 제목
    * 작성자
    * 비밀번호
    * 내용
* 게시물 삭제
    * 비밀번호
* 댓글
    * 댓글 작성
    * 댓글 삭제
* 파일
    * 파일 업로드
    * 파일 다운로드
    * 파일 삭제
    * 파일 수정
